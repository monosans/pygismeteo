# pygismeteo

[![Downloads](https://static.pepy.tech/badge/pygismeteo)](https://pepy.tech/project/pygismeteo)

Асинхронная версия [здесь](https://github.com/monosans/aiopygismeteo).

```bash
python -m pip install -U pygismeteo pygismeteo-base
```
